KAcademyX Educational Platform

KAcademyX is a comprehensive educational platform designed for online courses, instructor management, and automated certificate generation. Built with PHP and Bootstrap, it offers a seamless learning experience for students, instructors, and administrators alike.

------------------------------------------------------------

Features

- Course Management System
  Browse, enroll, and manage a variety of courses.

- Instructor Profiles
  Detailed profiles showcasing instructor expertise and courses.

- Certificate Generation
  Automated creation of certificates upon course completion.

- Resource Library
  Access to a wide range of learning materials and resources.

- Event Management
  Stay updated with educational events, webinars, and schedules.

- Pricing Plans
  Multiple subscription options tailored to user needs.

- Admin Panel
  Full administrative control over users, courses, and content.

- Responsive Design
  Optimized for all device sizes using Bootstrap.

------------------------------------------------------------

Technology Stack

- Frontend: HTML5, CSS3, JavaScript, Bootstrap
- Backend: PHP
- Database: MySQL
- Server: XAMPP (Apache, MySQL, PHP)
- Additional: TCP/IP protocols for specific functionalities, TCPDF for certificate generation

Installation

1. Install XAMPP
   Download and install XAMPP from https://www.apachefriends.org/

2. Clone the Repository
   Open your terminal or command prompt and run:
   cd C:\xampp\htdocs\
   git clone https://github.com/ahmed9088/KAcademyX.git

3. Start Services
   Launch Apache and MySQL from the XAMPP control panel.

4. Import Database
   If a database dump is provided, import it via phpMyAdmin (http://localhost/phpmyadmin).

5. Configure Database Connection
   Update the database connection settings in the application configuration files to match your environment.

6. Access the Application
   Open your browser and navigate to:
   http://localhost/KAcademyX
